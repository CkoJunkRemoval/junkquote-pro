# Success Metrics

A first-time user should complete their first estimate in under 10 minutes.

An experienced user should complete most estimates in under 5 minutes.

Every screen should reduce typing whenever possible.

Every workflow should minimize taps and clicks.

The software should always remember previously entered information whenever it makes sense.
